## Overview

This vlog demonstrates my full stack project "Student Task Manager" built using the MERN stack (MongoDB, Express, React, Node.js).

In this video, I explain the complete working of the application including frontend, backend, authentication, and database integration.

## What You Will Learn

- Project architecture and folder structure
- React components, props, and hooks (useState, useEffect)
- Protected routes using JWT
- Backend controllers and middleware
- MongoDB schema design
- Authentication and security implementation
- My learning experience during Vinternship
