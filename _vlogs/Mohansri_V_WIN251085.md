## About this Vlog

In this video, I demonstrate a basic backend server built using Express.js for the Greenfield Community scenario, where the goal was to move community activities like events and classes from manual handling to an online system.


Through this project, I built a simple backend server using Express.js that manages:

- Upcoming events
- Available classes
- Contact information
- Static file serving

This project helped me understand how backend servers handle routing, middleware, and requests in a structured way.
