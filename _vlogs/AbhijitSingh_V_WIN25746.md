## 🎥 Generics? Why?

In this video, I explain **why Generics are one of the most powerful features in TypeScript** and why every developer should understand them.

We start with a simple problem — a function that only works with numbers — and see how it fails when we try to use it with strings. From there, we explore how Generics solve this issue while keeping full type safety.

### In this video, you'll learn:

- The problem with hardcoded types
- Why using `any` is dangerous
- What Generics actually are
- How `<T>` works
- How TypeScript infers generic types
- A practical `ApiResponse<Data>` example
- Why Generics make code reusable and flexible

## 💡 Key Takeaway

Generics allow us to write reusable, flexible, and type-safe code without sacrificing developer experience.

Instead of writing multiple versions of the same function, we write it once — and let TypeScript handle the rest.
