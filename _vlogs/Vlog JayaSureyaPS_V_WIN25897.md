## About This Vlog

In this vlog, I explain how I built a Bakery API using Express, routing-controllers, and TypeScript as part of my Vinternship journey.

I demonstrate how to structure a backend project using controllers, how to define data types using TypeScript interfaces, and how to implement RESTful API endpoints. The project includes creating new orders, retrieving all orders, fetching a specific order by ID, starting the baking process, and checking the baking status of an order.

This video reflects my learning experience, the structure I followed, and how I separated responsibilities between the OrderController and BakingController to keep the backend modular and organized.
