## Project Overview

In this video, I break down the architecture and implementation of a **React Counter Application**. This project serves as a fundamental exercise in understanding how modern web applications manage data and user interactions.

### What You Will Learn:
* **Project Entry Points:** How `index.html` and `main.jsx` work together to initialize a React application using Vite.
* **Component-Based Architecture:** The benefits of modularizing code by importing the `Counter` component into a top-level `App.jsx`.
* **State Management:** How to use the `useState` hook in `Counter.jsx` to track and update the count.
* **Event Handling:** Implementing `increment`, `decrement`, and `reset` logic tied to button click events.
* **Styling with CSS:** Using Flexbox and HSL color values in `index.css` to create a responsive and visually appealing interface.

---

## Technical Deep Dive

The application is built using a clean separation of concerns. The logic resides within the `Counter` function, while the visual presentation is handled by a dedicated CSS file that centers the content and provides interactive hover states for the buttons.
